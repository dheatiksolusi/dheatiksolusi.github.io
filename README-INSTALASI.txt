CARA UNGGAH MELALUI GITHUB:
A. Tambahkan artikel baru
   - Buka folder blog di repository.
   - Klik Add file > Create new file.
   - Isi nama file:
     transformasi-digital-umkm-temanlaris-ai.html
   - Salin seluruh isi file artikel dari paket ini.
   - Klik Commit changes.

B. Ganti halaman daftar blog
   - Buka blog/index.html.
   - Klik ikon pensil (Edit this file).
   - Hapus isi lama, lalu tempel isi blog/index.html dari paket ini.
   - Klik Commit changes.

C. Ganti sitemap
   - Buka sitemap.xml di root repository.
   - Klik ikon pensil.
   - Hapus isi lama, lalu tempel isi sitemap.xml dari paket ini.
   - Klik Commit changes.

D. Pemeriksaan setelah deployment
   - Tunggu GitHub Pages sekitar 1–5 menit.
   - Buka:
     https://dheatiksolusi.my.id/blog/
   - Buka artikel:
     https://dheatiksolusi.my.id/blog/transformasi-digital-umkm-temanlaris-ai.html
   - Pastikan kelima tautan wajib dapat diklik:
     IDwebhost
     domain murah
     hosting murah
     PANDI
     domain .id

E. Agar masuk Google
   - Buka Google Search Console untuk dheatiksolusi.my.id.
   - Gunakan URL Inspection.
   - Masukkan URL artikel.
   - Klik Request Indexing.
   - Kirim ulang sitemap:
     https://dheatiksolusi.my.id/sitemap.xml

CATATAN:
- Artikel sudah lebih dari 2.000 kata.
- Artikel menggunakan aset visual berbasis CSS, sehingga tidak memakai gambar pihak lain.
- Jangan hapus artikel setelah dipublikasikan karena termasuk syarat lomba.
- Pastikan data penulis, tanggal, dan pengalaman pada artikel sudah sesuai sebelum submit.
